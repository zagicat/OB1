## Contribution Types

| Emoji | Type | Description |
| ----- | ---- | ----------- |
| 💻 | code | Code contributions (recipes, schemas, dashboards, integrations) |
| 📖 | docs | Documentation |
| 💡 | ideas | Ideas and use cases |
| 🧪 | testing | Testing contributions and reporting results |
| 🤝 | mentoring | Helping non-technical contributors shape their ideas into PRs |
| 👀 | review | Reviewing pull requests |
| 🐛 | bug | Bug reports |
| 🚧 | infra | Infrastructure, CI/CD, repo tooling |

Want to be added? Open a PR or contribute through any of the paths described in [CONTRIBUTING.md](CONTRIBUTING.md).
